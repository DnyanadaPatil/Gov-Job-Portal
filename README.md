# Government Job Portal

A full-stack web application for Maharashtra government job recruitment. Built with Python Flask backend and vanilla HTML/CSS/JavaScript frontend.

## Features

- User and Admin login system
- Browse available government job listings
- Apply for jobs with a simple form
- Admin panel to manage jobs and applications

## Tech Stack

- **Backend:** Python, Flask
- **Frontend:** HTML, CSS, JavaScript

## How to Run

1. Clone the repository
2. Install dependencies:
   ```
   pip install flask flask-cors
   ```
3. Navigate to the backend folder:
   ```
   cd backend
   ```
4. Start the server:
   ```
   python app.py
   ```
5. Open browser and go to `http://localhost:3000`
